## Categorias de Temas

### Temas Estaduais (compatível versão BaseV1)
- **theme-RS**: Customização para o estado do Rio Grande do Sul
- **theme-SpCultura**: Customização para o portal SP Cultura do estado de São Paulo
- **mapas-AC**: Customização para o estado do Acre
- **mapas-MA**: Customização para o estado do Maranhão
- **mapas-TO**: Customização para o estado do Tocantins
- **mapas-RR**: Customização para o estado de Roraima

### Temas Federais (compatível versão BaseV2)
- **theme-CulturaViva**: Tema para o programa Cultura Viva do Governo Federal
- **theme-MapaMinC**: Tema oficial do Ministério da Cultura
- **theme-Funarte**: Tema customizado para a Fundação Nacional de Artes

### Temas Especializados (compatível versão BaseV2)
- **theme-Museus**: Tema específico para sistemas estaduais de museus

### Temas Municipais (compatível versão BaseV2)
- **theme-Blumenau**: Customização para Blumenau/SC
- **theme-JoaoPessoa**: Customização para João Pessoa/PB
- **theme-SantoAndre**: Customização para Santo André/SP
- **theme-SaoJose**: Customização para São José/SC
- **theme-Sobral**: Customização para Sobral/CE

## Estrutura de Temas

Os temas do Mapas Culturais seguem uma estrutura padrão que permite:

- **Customização visual**: Cores, tipografia, layout e elementos visuais
- **Funcionalidades específicas**: Recursos adaptados às necessidades locais
- **Conteúdo personalizado**: Textos, imagens e informações específicas da região
- **Integração com sistemas locais**: APIs e serviços específicos do município/estado

## Como Usar um Tema

Para utilizar um tema em sua instalação do Mapas Culturais:

1. **Clone o repositório** do tema desejado
2. **Configure o tema** no arquivo de configuração da aplicação
3. **Personalize** conforme necessário para sua implementação
4. **Teste** todas as funcionalidades antes de colocar em produção

## Desenvolvendo um Tema Customizado

Para criar um novo tema:

1. **Use o theme-BaseV2** como ponto de partida
2. **Customize** os arquivos de template, CSS e JavaScript
3. **Adapte** as funcionalidades às necessidades específicas
4. **Documente** as customizações realizadas
5. **Teste** em diferentes dispositivos e navegadores

## Contribuindo

Se você desenvolveu um tema para Mapas Culturais e gostaria de incluí-lo nesta lista:

- Abra uma **issue** no repositório oficial
- Envie um **pull request** com a documentação do tema
- Certifique-se de que o tema segue as **boas práticas** de desenvolvimento
- Inclua **documentação** adequada sobre instalação e uso
