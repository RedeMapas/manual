# Mapas Culturais Plugins Index

Este documento fornece um índice de plugins desenvolvidos para Mapas Culturais, extraído da organização GitHub.

## Plugins

- [mapasculturais/plugin-RegistrationPayments](https://github.com/mapasculturais/plugin-RegistrationPayments): Adiciona a Entidade RegistrationPayments\\Payment para gerenciar pagamentos
- [mapasculturais/plugin-AldirBlanc](https://github.com/mapasculturais/plugin-AldirBlanc): Plugin que implementa no Mapas Culturais os formulários do inciso I e II da Lei Aldir Blanc
- [mapasculturais/plugin-AldirBlancDataprev](https://github.com/mapasculturais/plugin-AldirBlancDataprev): Plugin de exportação/importação das inscrições da Lei Aldir Blanc para o Dataprev
- [mapasculturais/plugin-AldirBlancValidador](https://github.com/mapasculturais/plugin-AldirBlancValidador): Plugin Validador Genérico
- [mapasculturais/plugin-AldirBlancValidadorFinanceiro](https://github.com/mapasculturais/plugin-AldirBlancValidadorFinanceiro): Plugin validador financeiro para Lei Aldir Blanc
- [mapasculturais/plugin-Metabase](https://github.com/mapasculturais/plugin-Metabase): Plugin que faz o embed dos dashboards do metabase no MapasCulturais
- [mapasculturais/plugin-Accessibility](https://github.com/mapasculturais/plugin-Accessibility): Plugin de acessibilidade para o sistema Mapas do Ceará
- [mapasculturais/plugin-SpamDetector](https://github.com/mapasculturais/plugin-SpamDetector): Plugin para detecção de spam
- [mapasculturais/plugin-AldirBlancValidadorRecurso](https://github.com/mapasculturais/plugin-AldirBlancValidadorRecurso): plugin Validador de Recursos para a Aldir Blanc
- [mapasculturais/plugin-Analytics](https://github.com/mapasculturais/plugin-Analytics): Faz o embed do google Analytics no MapasCulturais
- [mapasculturais/plugin-Zammad](https://github.com/mapasculturais/plugin-Zammad): Plugin que integrar o chat Zammad ao MapasCulturais
- [mapasculturais/plugin-AppealValidator](https://github.com/mapasculturais/plugin-AppealValidator): Plugin validador de recursos
- [mapasculturais/plugin-LocationPatch](https://github.com/mapasculturais/plugin-LocationPatch): Plugin para aplicar correções à geolocalização de agentes e espaços na medida que o Mapas é acessado pelo público
- [mapasculturais/plugin-DataAnonymization](https://github.com/mapasculturais/plugin-DataAnonymization): Plugin de Anonimização de Dados
- [mapasculturais/Plugin-FinancialValidador](https://github.com/mapasculturais/Plugin-FinancialValidador): Plugin que agrega funcionalidade de gerenciamento dos pagamentos
- [mapasculturais/plugin-MapasBlame](https://github.com/mapasculturais/plugin-MapasBlame): plugin para registro de logs do sistema
- [mapasculturais/plugin-MapasOportunidade](https://github.com/mapasculturais/plugin-MapasOportunidade): Plugin que promove uma oportunidade na interface, facilitando o acesso do usuário
- [mapasculturais/plugin-MapasOportunidadeFrontend](https://github.com/mapasculturais/plugin-MapasOportunidadeFrontend): Plugin frontend para oportunidades
- [mapasculturais/plugin-EmbedTools](https://github.com/mapasculturais/plugin-EmbedTools): Plugin que possibilita embutir funções do thema BaseV1 no Thema Basev2
- [mapasculturais/plugin-GenericValidator](https://github.com/mapasculturais/plugin-GenericValidator): Plugin com um validador genérico instanciável mais de uma vez
- [mapasculturais/plugin-AccountConsolidator](https://github.com/mapasculturais/plugin-AccountConsolidator): Plugin para consolidação de contas
- [mapasculturais/plugin-MailImage](https://github.com/mapasculturais/plugin-MailImage): Plugin para manipular as imagens dos e-mails enviados
- [mapasculturais/plugin-SubsiteFilter](https://github.com/mapasculturais/plugin-SubsiteFilter): Plugin que filtra a api dos subsites para retornar somente entidades criadas no subsite
- [mapasculturais/plugin-GeodivisionsBR](https://github.com/mapasculturais/plugin-GeodivisionsBR): pugin para popular a tabela geo_divisions com estados, macrorregiões, microrregiões e municípios do Brasil
- [mapasculturais/plugin-ClaimForm](https://github.com/mapasculturais/plugin-ClaimForm): Plugin que possibilita a solicitação de recurso com anexo
- [mapasculturais/plugin-DisableModules](https://github.com/mapasculturais/plugin-DisableModules): Plugin para desligar módulos do MapasCulturais
- [mapasculturais/plugin-SubsiteImport](https://github.com/mapasculturais/plugin-SubsiteImport): Plugin para importação das entidades de subsites
- [mapasculturais/plugin-Integrator](https://github.com/mapasculturais/plugin-Integrator): Plugin integrador de instalações de Mapas Culturais
- [mapasculturais/plugin-EventActions](https://github.com/mapasculturais/plugin-EventActions): plugin que adiciona as funcionalidades de "comparecerei" e "tenho interesse" aos eventos
- [mapasculturais/plugin-HomeContent](https://github.com/mapasculturais/plugin-HomeContent): Publigin para inserir conteúdo na home principal
- [mapasculturais/plugin-ValuersManagement](https://github.com/mapasculturais/plugin-ValuersManagement): plugin para gerenciar os avaliadores por planilhas importadas
- [mapasculturais/plugin-MetadataKeyword](https://github.com/mapasculturais/plugin-MetadataKeyword): plugin para definir keywords nos campos que estão nas tabelas de metadado Ex.: AgenteMeta
- [mapasculturais/plugin-AbstractValidator](https://github.com/mapasculturais/plugin-AbstractValidator): Plugin que agrega funcionalidades gerais de validação para uso pelos demais validadores
- [mapasculturais/plugin-UndeletableEntities](https://github.com/mapasculturais/plugin-UndeletableEntities): Plugin para entidades não deletáveis
- [mapasculturais/plugin-AldirBlancRedirects](https://github.com/mapasculturais/plugin-AldirBlancRedirects): Redireciona o usuário para o cadastro e mantém o Mapas fechado para usuários comuns
- [mapasculturais/plugin-CorrecaoRepresentanteLegal](https://github.com/mapasculturais/plugin-CorrecaoRepresentanteLegal): Plugin que corrige a base de dados quebrando os diversos agentes individuais em várias contas
- [mapasculturais/plugin-GenericPaymentExport](https://github.com/mapasculturais/plugin-GenericPaymentExport): Implementa plugin genérico para exportação de dados financeiro
- [mapasculturais/plugin-CreateGeoDivisions](https://github.com/mapasculturais/plugin-CreateGeoDivisions): Plugin para criação de divisões geográficas

## Como Usar

Para utilizar qualquer um destes plugins em sua instalação do Mapas Culturais, consulte a documentação específica de cada plugin no respectivo repositório GitHub. A maioria dos plugins segue o padrão de instalação através do composer ou download direto dos arquivos.
