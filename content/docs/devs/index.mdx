O projeto Mapas Culturais é uma plataforma para *mapear e conectar* agentes, espaços, eventos, projetos e oportunidades culturais. Ele permite que usuários cadastrem suas **Entidades** culturais (como artistas, locais, shows), formando um mapa interativo e uma base de dados cultural. A plataforma também gerencia **Oportunidades** (editais) com seus processos de inscrição e **Avaliação**. Usuários interagem com a plataforma através de **Componentes Vue** modernos, que se comunicam com o servidor usando uma **Classe API** centralizada. Funcionalidades importantes incluem uma **Busca** avançada com filtros, um **Painel de Controle** pessoal para gerenciar conteúdos e tarefas, e um sistema de **Autenticação** flexível.

**Código-fonte:** [Redemapas/mapas](https://github.com/redemapas/mapas)
